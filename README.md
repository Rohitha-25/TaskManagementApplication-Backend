# 📝 Task Management Application (Backend)

## Overview
A backend service built using Spring Boot to manage tasks with RESTful APIs, supporting CRUD operations and integration with frontend applications.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Hibernate

## Features
- CRUD operations for tasks
- REST API endpoints
- Database integration
- Layered architecture (Controller → Service → Repository)

## Key Concepts
- Dependency Injection
- JPA & ORM mapping
- REST API design
